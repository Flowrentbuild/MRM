# MRM Menuiserie — Console de contrôle des bons de commande

Console web brandée MRM qui dépose les pièces (devis, métré, commande MRM ou ARC fournisseur)
vers le workflow n8n de contrôle automatique. Le rapport est renvoyé par email.

Production : https://mrm-verificateur.flowrentbuild.workers.dev/

## Contenu

- `index.html` — page statique (HTML/CSS/JS dans un seul fichier)
- `README_deploiement.md` — guide de déploiement Cloudflare Pages

## Modes

- **Single** : un fournisseur, contrôle d'une commande/ARC unique
- **Multi** : un chantier multi-fournisseurs, plusieurs commandes/ARC en une fois
